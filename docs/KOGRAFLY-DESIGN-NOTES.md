# Kografly Design Notes

Kografly mengambil inspirasi visual dari ThreadWeave tanpa menyalin konsep forum/discussion-nya secara langsung.

## Adaptasi visual

- Indigo `#4338CA` untuk primary CTA dan link aktif.
- Amber `#D97706` untuk highlight kecil seperti slash `/` dan label.
- Teal `#0F766E` untuk realtime/connection state.
- Stone background `#FAFAF9` dan white surface untuk card.
- Crimson Text untuk heading editorial.
- Motif "thread" dipakai sebagai aksen vertical indent 24px pada beberapa card, bukan nested forum replies.

## Identitas Kografly

- Bentuk UI lebih dekat ke bio-link/creator dashboard.
- Komponen utama: public bio page, builder, preview phone, dashboard analytics.
- Style button punya varian solid/outline/soft/glass.
- Animasi per link dibuat sebagai microinteraction creator-oriented.
